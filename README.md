# Backend Api Todo List

## Endpoints
**URL Base** : [https://todoapp-back.herokuapp.com/api/v1/](https://todoapp-back.herokuapp.com/api/v1/)

### GET /todos
Seleciona todos as tarefas criadas

### POST /todos
Cria uma nova tarefa

### PUT /todos/:id
Edita uma tarefa

### DELETE /todos/:ID
Delete uma tarefa
